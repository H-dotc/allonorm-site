Aperçu d'accessibilité de votre site

Analyse réelle de la page d'accueil de votre site : nous téléchargeons son code et y détectons les principaux défauts d'accessibilité, comptés un par un. Le diagnostic complet (toutes pages, tous critères) est gratuit, sur demande.

⚠️ Ces outils sont indicatifs. Le simulateur se fonde sur les seuils légaux officiels et les éléments que vous saisissez ; les montants affichés sont des plafonds réglementaires, non un calcul de votre amende réelle. L'aperçu analyse réellement le code de votre page d'accueil, mais ne couvre qu'une partie des 106 critères RGAA (~30 % sont automatisables). Ils ne constituent ni un audit, ni un avis juridique : seul un audit RGAA complet établit votre conformité réelle.
